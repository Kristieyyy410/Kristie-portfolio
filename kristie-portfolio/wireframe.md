# Portfolio Wireframe

## 1. Page Structure

1. Sticky top navigation
2. Hero section with bilingual positioning
3. Key metrics strip
4. About me section
5. Selected projects cards
6. AI workflow section with prompt example
7. Experience timeline
8. Social media section
9. Lifestyle and fitness section
10. Contact call-to-action

## 2. Desktop Layout

- Hero: 2-column split
  - Left: title, lead, CTA, metrics
  - Right: portrait image + focus note
- Metrics: 4-column stat cards
- About: text card + pill tags
- Projects: 3 equal cards
- AI workflow: 3 cards + prompt box
- Experience: vertical timeline
- Social: copy card + image gallery
- Lifestyle: image-led cards with fitness emphasis
- Contact: compact CTA panel

## 3. Mobile Layout

- Topbar wraps into two lines
- Hero stacks vertically
- Metrics collapse to one column
- Cards become single-column blocks
- Gallery becomes vertical stack

## 4. Visual Direction

- Background: warm ivory
- Text: charcoal black
- Accent: champagne gold
- Typography: editorial serif for headings, modern sans for body
- Style: minimal, polished, quiet luxury

## 5. Content Priority

1. AI growth positioning
2. Quantified project outcomes
3. Chinese/English credibility
4. AI workflow differentiation
5. Personal brand and fitness identity
