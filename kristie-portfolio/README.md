# Kristie Zheng Portfolio

Static bilingual portfolio site for AI growth, community operations, content growth, and creator operations.

## Files

- `index.html` - single-page portfolio shell
- `styles.css` - visual system and responsive layout
- `content.js` - bilingual content dictionary
- `app.js` - language toggle, rendering hooks, and scroll reveal
- `assets/` - portfolio images

## Run locally

Open `index.html` directly in a browser, or serve the folder with a local static server.
